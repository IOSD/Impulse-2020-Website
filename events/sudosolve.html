SUDO SOLVE

Feel like participating in a hardcore code event? We've got a two-day coding event for all you geeks out there! Four engaging sub events adaptandattack, costlycharacters, bugbusters and competitive programming and you’d still go home wanting for more!

Venue : Maharaja Agrasen Institute Of Technology

Date and Time : 31 March 10:00 AM

Registration Deadline : 25 March 11:59 PM

Rules

This event comprises of four subevents, description of which are given below. In order to win this event, a participant must go through all the subevents.

AdaptANDAttack

Shouting out to all the code geeks. Here, at the very first level as the title suggests the participants will be given the details of the programming language on the spot using which they have to solve the given questions.

Only after successfully solving a question you can move onto the next one. Top solutions will move onto the next round..

CostlyCharacters

In this event you have to solve the given question and also write the shortest code possible for it. Solve the problems, optimize your code and shorten it. The shortest solution will move on to the next event.

BugBuster

In this round, you'll have to find and eliminate bugs from a given code snippet.

Each kill will help you win the round.

Competitive Programming

The flagship event of the SUDOSOLVE. In this round, participants will be given questions based on DSA.

On successful solution of the given problem the participant will be pronounced as the ultimate winner and would end up winning SUDOSOLVE

FAQ's
Q) Can the students bring their own laptop for the event?

No, as the competition would be held in computer labs where each participant would be provided with a computer; no laptops are allowed during this event.

Q) How will the participants be assessed while debugging the given code?

Number of bugs found and eliminated. Bonus Marks for improved code complexity

Q) Codes given for debugging will be in which language?

Language used will be C++.

Q) In the competitive programming event, what languages can be used?

Participants can write code in Python, C++ or Java.
